# spiderMan

> 工具需在**python3**环境下

```
pip install -r requirements.txt 
```
>spiderman/plugin/config 下page = 40 设置爬取搜索的的页数 影响最后的结果数量

## 设置关键词，例子：刷赞网、刷钻网、代刷平台、24小时自助代刷平台、QQ 代刷网、低价QQ刷钻、免费QQ刷赞网\低价名片赞

```
python3 spiderMan.py --keyword inurl:刷赞网 run
```
